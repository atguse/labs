Difference between block and inline

This is a span This is another span

Examples of block-level elements:

  • < div >
  • < h1 > - < h6 >
  • < p >
  • < form >
  • < header >
  • < footer >
  • < section >

Examples of inline elements:

  • < span >
  • < a >
  • < img >

can you display an inline element like a block one?

yes, use the css property called display
You can see here some example in order to fully understand how to use display.

A block-level element always starts on a new line and takes up the full width available (stretches out to the left and right as far as it can).
An inline element does not start on a new line and only takes up as much width as necessary.

Difference between margin and padding

This is a div with 100px padding
This is a div with 100px margin

Padding is the space between the content and the border, whereas margin is the space outside the border.
